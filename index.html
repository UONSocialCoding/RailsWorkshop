Writing production level code
Production is scary!
  • Real people will use your code
  • A small bug can have catastrophic consequences
  • Security vulnerabilities can do real damage
How can we mitigate the badness?
Observe! The 7 commandments
1. Write tests and run them before committing
  1. Write tests!
  2. Write code in branches
  3. Use GitHub boards
  4. Use continuous integration
  5. Use code quality tools
  6. Review each others code
  7. Have a staging environment
